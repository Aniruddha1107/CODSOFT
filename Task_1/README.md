This is 'the Number Guess Game'.
The System gives a random number from 0 to 100 using Random Class.
User is required to guess the number.
User is given 5 attempts to do so.
When user enter high number then the guess number then a message is shown as 'too high, try low', so, user has to give a low number.
And vise-versa when low number is entered.
When correct answer is given user gets a point.
A choice is given to user to play again or quit.
When user quits, a final score is displayed.
